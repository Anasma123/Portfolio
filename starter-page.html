Calculator

Description: This project is a fully functional calculator application designed to handle both basic and advanced mathematical operations. It supports addition, subtraction, multiplication, division, as well as scientific functions like square root, trigonometry (sine, cosine, tangent), and logarithms. Task Management API

Description: The Task Management API is a RESTful backend service designed to power a task management application. It allows users to create, read, update, and delete tasks, as well as manage task categories, deadlines, and priorities. The API supports user authentication using JWT (JSON Web Tokens) to ensure secure access to user-specific data. It integrates with MongoDB for persistent storage, utilizing Mongoose for schema validation and efficient querying. The project aims to provide a scalable backend solution that can be integrated with various frontend applications, offering features like task filtering, sorting, and pagination to enhance usability.

Languages: Node.js, Express, MongoDB

Technologies: REST API, JWT Authentication, Mongoose

Purpose: Backend

Duration: 1 month

Challenges: Ensuring secure authentication and authorization mechanisms to protect user data, optimizing database queries for large datasets, and implementing proper error handling for a robust API. Another challenge was setting up CORS to allow cross-origin requests from frontend applications securely.

Status: Ongoing

Fitness Tracking App

Description: The Fitness Tracking App is a mobile application built to help users track their fitness activities, set goals, and monitor progress over time. The app allows users to log workouts (e.g., running, cycling, weightlifting), track calories burned, and set daily step goals. It integrates with Google Fit API to sync fitness data from wearable devices and uses Firebase for real-time data storage and user authentication. The app features a dashboard with visual charts to display progress, a workout planner with reminders, and a social feature to share achievements with friends. The goal is to create an all-in-one fitness solution that motivates users to stay active and achieve their fitness goals.

Languages: Flutter, Dart

Technologies: Firebase, Google Fit API, Charts

Purpose: Frontend & Backend

Duration: 3 months

Challenges: Integrating with the Google Fit API to fetch accurate fitness data, optimizing the app for performance on low-end devices, and ensuring real-time synchronization with Firebase without excessive battery drain. Designing an intuitive UI that caters to both beginners and advanced users was also a significant challenge.

Status: On Progress
